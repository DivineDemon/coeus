# Clearbeam

## Snapshot

Clearbeam is a SaaS analytics and alerting product for founders and small teams. It combines event ingestion, dashboard analytics, quota management, and billing into a deployable multi-tenant product.

## Stage

- Stage: active product with production deployment and monetization rails.
- Focus: help teams track critical business events quickly with reliable alerts.

## What exists today

- Full app described in `[[03-resources/github-repos/clearbeam|clearbeam]]` with event API, analytics, and settings workflows.
- Production deployment is listed in the repo note and Vercel project inventory.
- Paid plan enablement through Stripe and transactional communication through Resend.

## Core capabilities

- API-key-based event ingestion (`POST /api/v1/events`).
- Real-time and historical analytics views.
- Subscription billing, usage quotas, and account management.
- SaaS-friendly stack using Next.js, TypeScript, Prisma, Neon, and Stripe.
