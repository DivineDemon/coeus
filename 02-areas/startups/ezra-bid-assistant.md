# Ezra Bid Assistant

## Snapshot

Ezra Bid Assistant is an AI-assisted proposal drafting product for Freelancer.com workflows. It combines a Chrome extension UX with a backend generation API so users can generate and insert proposals faster while keeping manual submission control.

## Stage

- Stage: active build with deployed backend and private extension workflow.
- Positioning: productivity and conversion assistant for freelance bidding.

## What exists today

- Product architecture is documented in `[[03-resources/github-repos/ezra-bid-assistant|ezra-bid-assistant]]` (backend + extension + shared workspace).
- Resume evidence ties this venture to production deployment and brand alignment (`ezraglobal.co.za`).
- Related private deployment projects include `ezra-bid-assistant-backend` and `ezra-global`.

## Core capabilities

- Page-aware extraction of visible Freelancer project details.
- Proposal generation via secured backend and Gemini model configuration.
- Human-in-the-loop insertion into the bid textbox with no auto-submit behavior.
- Local draft saving, style/length controls, and backend connection testing.
