# Coeus

> My 2nd Brain. (Obsidian)

This is the repository you are reading right now — an Obsidian vault organized with the PARA method (Projects, Areas, Resources, Archives) that serves as a personal knowledge base and career second brain.

## Purpose

- Snapshot and cross-link GitHub repos, social profiles, startup ventures, and deployed projects
- Maintain a skills matrix derived from real evidence (repos, resumes, deployments)
- Track career timeline, resume versions, and conflict audits
- Provide a graph-navigable map of all professional assets

## Structure

```
00-inbox/           # Unsorted captures
01-projects/        # Active deployments and initiatives
02-areas/           # Ongoing responsibilities (Career, Startups, Immigration)
03-resources/       # Reference material (Repos, Resumes, Skills, Social Profiles)
04-archives/        # Completed or inactive items
templates/          # Note templates for repos, startups, profiles, projects
```
